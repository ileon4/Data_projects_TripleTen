# Data_projects_TripleTen
These projects were completed during the course of my data science bootcamp and demonstrate my skills in Python data cleaning and analysis as well as Machine Learning.

| Project # | Title | Description |
|-----------|-------|-------------|
| 1 | Customer Churn Predictive Modeling | This project analyzes customer churn for a bank and addresses class imbalance and the training and testing of various models |
| 2 | Telephone Provider Customer Behavior Model Analysis | I develop a model for a mobile carrier that analyzes subscribers' behavior. I look for any errors in the data, compare models, and compare their accuracy. |
| 3 | Video Game Marketing Analysis | I analyze video game sales data to identify patterns that determine a game's success |
| 4 | Telecom Marketing Analysis | Using Pandas EDA and matplotlib visualizations, I analyze the clients' behavior of a telecom operator company and determine which prepaid plan brings in more revenue |
